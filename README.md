Запуск проекта
1. Колировать репозиторий локально 
2. Перейти в директорию 
3. python3 -m venv venv
4. source venv/bin/activate
5. pip install -r requirements.txt
6. python manage.py runserver

base.html - база

http://127.0.0.1:8000/ask/ - станица добавления вопроса

http://127.0.0.1:8000/ - начальная страница

http://127.0.0.1:8000/login/ - страница входа

http://127.0.0.1:8000/question/{n}/ - страница конкретного вопроса

http://127.0.0.1:8000/signup/ - страница регистрации

http://127.0.0.1:8000/profile/ - настройки аккаунта

http://127.0.0.1:8000/tag/{name}/ - страница с вопросами по тегу
