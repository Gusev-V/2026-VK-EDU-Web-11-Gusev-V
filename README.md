Запустить верстку можно склонировав ветку локально, после его открывать страницы через любой браузер как файл (при открытие в safari каким образом может пропасть фавикон), либо запустить локальный сервер и открыть через него (например: открыть проект (папка public) в VS Code, скачать если необходимо расширение Live Server, и запустить его). Список станиц: 
* ask.html - станица добавления вопроса 
* base.html - база 
* index.html - начальная страница 
* question.html - страница конкретного вопроса 
* reg.html - страница регистрации 
* setting.html - настройки аккаунта 
* tags.html - страница с вопросами по тегу
